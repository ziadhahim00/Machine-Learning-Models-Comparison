# Machine Learning Models Comparison

## Overview
This project compares multiple machine learning models across different datasets for both regression and classification tasks.

## Datasets Used
- Insurance Dataset (Regression)
- Student Performance Dataset (Regression)
- Heart Disease Dataset (Classification)
- Diabetes Dataset (Classification)
- Mushroom Dataset (Classification)
- Wine Dataset (Classification)

## Models Used
### Regression
- Ridge Regression with different lambda values
- Cross-validation for model selection

### Classification
- Logistic Regression
- Neural Network (MLPClassifier)

## Steps
- Data preprocessing (encoding categorical variables)
- Feature scaling using StandardScaler
- Model training and evaluation
- Performance comparison using multiple metrics

## Evaluation Metrics
### Regression
- MSE
- RMSE
- MAE
- R² Score

### Classification
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Results
- Compared model performance across multiple datasets
- Neural Networks generally achieved higher accuracy
- Logistic Regression provided stable and interpretable results

## Visualization
- Accuracy comparison chart between Logistic Regression and Neural Network

## Conclusion
This project demonstrates the differences between regression and classification models and highlights how model performance varies across datasets.
